# A Tracker

一个简洁的 Chrome 扩展，用于追踪贵金属、加密货币和股票的实时价格。

## 功能

- 实时价格：通过 Yahoo Finance API 获取最新行情
- 盘前/盘后价格：支持显示盘前、盘中、盘后时段价格
- 角标显示：可将关注的品种价格展示在扩展角标上
- 紧凑模式：一行展示一个品种，适合追踪更多标的
- 自定义品种：支持添加、删除、排序关注的品种
- 主题切换：支持跟随系统、深色、浅色三种主题

## 默认品种

- GC=F（黄金）
- SI=F（白银）
- HG=F（铜）
- BTC-USD（比特币）
- ETH-USD（以太坊）

## 安装

1. 打开 Chrome 浏览器的扩展管理页面（`chrome://extensions/`）
2. 开启右上角的「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本项目目录

## 使用

- 点击浏览器工具栏的 A Tracker 图标，查看所有关注品种的实时价格
- 在设置页面中添加自定义品种、调整排序、切换主题和紧凑模式
- 使用快捷键 `Ctrl+K`（Mac 为 `Command+K`）快速展示/隐藏角标价格

## 数据来源

行情数据来自 [Yahoo Finance](https://finance.yahoo.com/)。
